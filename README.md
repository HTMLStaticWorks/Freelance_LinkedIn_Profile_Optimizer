# Freelance LinkedIn Profile Optimizer Template

Premium multipurpose HTML template focused on career branding, LinkedIn optimization services, and executive profile enhancement.

## Features
- **Modern & Premium Design**: ThemeForest-quality UI/UX.
- **Fully Responsive**: Mobile-first architecture using Bootstrap 5.
- **Dark & Light Mode**: Built-in support with persistent preference.
- **RTL Support**: Full Right-to-Left layout support for Arabic/Hebrew.
- **SEO Optimized**: Semantic HTML, meta tags, and JSON-LD structured data.
- **Fast Performance**: Optimized assets and minimal dependencies.

## Pages Included
- Home 1: Corporate Landing
- Home 2: Personal Branding
- About Us
- Services & Service Details
- Pricing
- Blog & Blog Details
- Contact Us
- Login & Register
- 404, Coming Soon, Maintenance

## Technology Stack
- HTML5 & CSS3
- Bootstrap 5 (v5.3.0)
- Vanilla JavaScript (ES6+)
- Lucide Icons
- Google Fonts (Poppins & Inter)

## Folder Structure
- `assets/css`: Core styles, dark mode, and RTL overrides (Bootstrap compatible).
- `assets/js`: Main interaction logic and toggles.
- `pages/`: (Note: Pages now in root)
- `documentation/`: Detailed setup guide.

## Setup Instructions
1. Open any HTML page in the root directory in your browser.
2. To customize colors, edit the CSS variables in `assets/css/style.css`.
3. Replace images in the `assets/images/` folder with your own branding.
